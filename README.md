# Python Random Module Practice 🎲

This repository contains **practice programs and exercises** for Python's `random` module.  
It includes real-world applications like games, lotteries, password generation, AI decisions, and probability logic.  

These exercises are perfect for **internship preparation** and strengthening **problem-solving skills** in Python.  

---

## Contents

1. **Number Guessing Game**  
   - Generate a random number (1–10)  
   - Check if user's guess is too high, too low, or correct  

2. **Random Name Selection**  
   - Select one random name from a list using `choice()`  

3. **Dice System**  
   - Roll 2 dice, calculate total, decide who wins  

4. **Generate 5 Unique Random Numbers**  
   - Numbers between 1–50  
   - Use `randint()` and `sample()` for non-repetition  

5. **Random Fruit Selection**  
   - Select 2 fruits **with repetition** (`choices()`)  
   - Select 2 fruits **without repetition** (`sample()`)  

6. **Game Probability Logic**  
   - "Win" or "Lose" with 80% probability using `weights`  

7. **Playlist System**  
   - Pick a random song from a playlist each time  

8. **Lottery System**  
   - Select 5 winners from 100 participants without repetition using `sample()`  

9. **Password Generator**  
   - Generate password with 2 uppercase letters, 2 numbers, 2 symbols  

10. **AI Decision System**  
    - AI chooses Attack (70%) or Defend (30%) using `weights`  

11. **Reproducible Random Outputs**  
    - Use `getstate()` and `setstate()` to get the same random result multiple times
